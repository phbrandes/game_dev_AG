# 05_ASSET_PIPELINE.md

## Production Workflow

Greybox
→ Placeholder
→ Functional Prototype
→ Concept Art
→ Production Asset
→ Polish

## Recommended Tools

UI:
- Google Stitch

Concept Art:
- AI image generation

Sprites:
- Aseprite
- Krita

3D Concepts:
- Blender

## AI Workflow

1. Generate moodboards
2. Generate concept sheets
3. Create asset bible references
4. Produce final assets
5. Import into Godot

## Naming Convention

building_miner_t1
building_furnace_t1
conveyor_basic
turret_gun_t1

## Import Rules

Keep:
- Source asset
- Exported asset
- Godot import preset

## Consistency Rule

No asset enters production unless it matches:
- Art Bible
- Palette
- Scale
- Camera angle
