# 02_GAME_DESIGN_DOCUMENT.md

## High Concept
A factory automation roguelike where players build industrial fortresses that must survive escalating threats.

## Core Inspirations
Factorio
Mindustry
Dyson Sphere Program

## Design Pillars

### The Factory Is The Hero
Optimization is the primary gameplay.

### Combat Pressures Logistics
Combat exists to stress production chains.

### Automation Is Reward
Manual work should gradually disappear.

### Visible Growth
Each run should feel dramatically stronger.

### Meaningful Decisions
Every building should introduce strategic tradeoffs.

## Core Loop

Explore
→ Extract Resources
→ Process Materials
→ Expand Factory
→ Generate Power
→ Defend Infrastructure
→ Unlock Technology
→ Survive Waves
→ Harvest Progression

## Resource Tiers

Tier 1:
- Iron
- Copper
- Coal

Tier 2:
- Steel
- Circuits
- Power Components

Tier 3:
- Advanced Electronics
- Robotics
- High-Energy Materials

## Building Categories
- Logistics
- Production
- Power
- Defense
- Research
- Storage

## Progression Philosophy
Avoid:
+2% upgrades

Prefer:
New machines
New systems
New logistics options
New strategies

## Enemy Design
Enemy pressure should create production problems.
