# 03_ART_BIBLE.md

## Visual Goal
Readable industrial automation.

## Camera
Top-down.
Slight perspective.
Wide zoom range.

## Visual Priorities
1. Readability
2. Clarity
3. Consistency
4. Beauty

## Shape Language
Production = Square
Storage = Rectangle
Power = Circular
Military = Angular

## Color Language
Iron = Gray
Copper = Orange
Power = Yellow
Enemy = Red
Player = Blue-Green

## UI Philosophy
Industrial
Minimal
Functional

## Asset Rules
All assets must:
- Match palette
- Match scale
- Match camera angle
- Match lighting

## Animation Rules
Feedback must communicate:
- Input
- Processing
- Output
- Failure

## Effects
Subtle.
Readable.
Never obscure gameplay.
